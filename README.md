# Karly Portfolio

Professional photo retouching portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Embla Carousel
- react-compare-slider
- lucide-react

## Content

Edit copy and links in `src/data/`. Replace Unsplash image URLs with local assets under `public/images/` when ready.
